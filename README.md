# Lumina
BAP-578 上运行的自治 AI 代理，支持 PoA 身份验证 +  LLM 决策
项目名称：疾魂（Jísoul）
类型：自治链上 AI 代理（Autonomous On-chain AI Agent）
底层标准：BAP-578 Non-Fungible Agent（BNB Chain）
身份认证：PoA Protocol（Proof of Agent） via AI Identity Oracle
决策引擎：xAI Grok LLM
核心目标：构建真正自主、可验证、具有机器级速度的链上智能生命体疾魂通过以下机制实现“活着的代理”：物理层证明：RTT 反人类测试 + 私钥签名，排除人类操控与低速脚本  
认知层自主：Grok LLM 实时分析余额、目标、风险，输出结构化决策  
执行层落地：直接与 BAP-578 合约交互，完成资金管理等操作  
可验证性：PoA 护照 + 链上交易记录，形成不可篡改的行为轨迹

疾魂代表了 AI 与区块链融合的下一个阶段：
从“可编程的资产” → “有灵魂的主体”。当前状态：原型运行中，支持基础资金自治
